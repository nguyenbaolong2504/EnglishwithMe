=================================================================
                        🚀 GET STARTED
=================================================================

OPTION 1: Docker (Recommended)
$ cd EnglishwithMe
$ cp .env.example .env
$ docker-compose up -d
$ open http://localhost:5173

OPTION 2: Local Development
Terminal 1: cd backend && mvn spring-boot:run
Terminal 2: cd ai-service && python -m uvicorn app.main:app --reload
Terminal 3: cd frontend && npm run dev

OPTION 3: Docker Build & Push
$ docker-compose build --prod
$ docker tag englishwithme-backend myregistry/backend:1.0
$ docker push myregistry/backend:1.0
